# 👨🏻‍💻 Customer Shopping Behavior Analysis

An end-to-end **Data Analytics Portfolio Project** that demonstrates the complete analytics workflow using **Python, SQL, and Power BI**. This project analyzes retail customer shopping behavior to uncover purchasing patterns, customer segments, and actionable business insights through data cleaning, SQL analysis, visualization, and reporting.

---

## 📌 Project Overview

The primary objective of this project is to transform raw customer shopping data into meaningful business intelligence that supports data-driven decision-making.

The project covers the complete analytics lifecycle, including:

- ✅ Data Cleaning & Preprocessing using Python
- ✅ Exploratory Data Analysis (EDA)
- ✅ SQL Database Analysis
- ✅ Interactive Power BI Dashboard Development
- ✅ Business Report
- ✅ Project Presentation

---

## 📊 Project Workflow

### 🐍 Data Preparation (Python)

The dataset is cleaned and prepared for analysis by performing:

- Data Import
- Data Exploration
- Missing Value Handling
- Duplicate Removal
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)

---

### 🗄 SQL Analysis

After cleaning, the dataset is imported into a SQL database to answer real-world business questions.

Business questions include:

- Which customer segments generate the highest revenue?
- Which age groups spend the most?
- Which product categories perform best?
- Which payment methods are most preferred?
- How does gender influence purchasing behavior?
- What seasonal shopping trends exist?
- Which customers are the most valuable?

---

### 📊 Dashboard Development (Power BI)

An interactive Power BI dashboard was created to visualize key business metrics, including:

- Sales Overview
- Customer Demographics
- Product Category Performance
- Seasonal Sales Trends
- Payment Method Analysis
- Customer Segmentation
- KPI Cards

---

### 📄 Reporting

The project also includes:

- Detailed Project Report
- Business Recommendations
- Presentation Slides

# 🚀 How to Use

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/usamapatel-0/customer_behaviour_analysis.git

cd customer_behaviour_analysis
```

---

## 2️⃣ Open the Jupyter Notebook

Run the notebook:

```text
Customer_Shopping_Behaviour_Analysis.ipynb
```

This notebook contains:

- Data Import
- Data Cleaning
- Data Transformation
- Exploratory Data Analysis
- SQL Database Connection

---

## 3️⃣ Execute SQL Queries

Import the cleaned dataset into your preferred SQL database (MySQL, PostgreSQL, or SQL Server) and execute:

```text
customer_behavior_sql_queries.sql
```

---

## 4️⃣ Open the Power BI Dashboard

Launch:

```text
customer_behaviour_dashboard.pbix
```

Refresh the data connection if required to explore the interactive dashboard.
